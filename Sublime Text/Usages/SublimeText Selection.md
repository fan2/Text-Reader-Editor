Sublime 的选择方式丰富而强大。

默认在光标所在行，按住 Command 键可点选其他行，实现选中多行。

复制粘贴当前行：【Cmd + Shift + D】复制当前行并粘贴到下一行。

## 块选择列编辑
Sublime Text 2 如何实现块选择列编辑，就是像 notepad++ 里面的一样，按住 alt 键，可以选择多个行的同一列。

Sublime 支持“多选区”，请注意与“块选”区别。

    Mac OS X：用alt/option

Windows：用 中键（或 Shift+右键）。

    用 Alt 配合，可从已有选区中挖减；
    用 Ctrl 配合，可增加独立选区。
