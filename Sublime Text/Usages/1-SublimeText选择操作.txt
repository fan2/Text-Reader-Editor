Sublime 的选择方式丰富而强大。

复制粘贴当前行：【Cmd + Shift + D】复制当前行并粘贴到下一行

Command + D：Expand Selection to Word
    基于分隔符（word_separators）选中当前光标所在的单词
    选中部分文本时，再次选中下一次出现的该文本
Command + L：Expand Selection to Line
    选中当前光标所在行，结合command＋D已选扩展到行，command＋delete可删除所有行。
Command + A：全选

按住Command键，再点击想选中的行：选中多行
Command + Control + G：选中部分文本时，再按此组合键，可选中所有相同文本

================================================================================
Sublime Text 2 如何实现块选择，就是像notepad++里面的一样，按住alt键，可以选择多个行的同一列。

Sublime 支持“多选区”，请注意与“块选”区别。
    Mac OS X：用alt/option

Windows：用 中键（或 Shift+右键）。
    用 Alt 配合，可从已有选区中挖减；
    用 Ctrl 配合，可增加独立选区。
